# MySQL

## 变种
* Drizzle：可用性增强，但很多地方不兼容
* MariaDB：Mysql之父在mysql被收购时创立，是Mysql的超集，完全兼容
* Percona：性能增强，《高性能Mysql》的作者创立
* Postgre：稳定性极强，抗压；数据类型丰富
* SQLite：小巧，嵌入式
