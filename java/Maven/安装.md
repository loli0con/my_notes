# 安装
[maven安装官方教程](https://maven.apache.org/install.html)

## 软件结构
* **bin**：含有Maven的运行脚本
* boot：含有plexus-classworlds类加载器框架
* **conf**：含有Maven的核心配置文件
* lib：含有Maven运行时所需要的Java类库
* LICENSE、NOTICE、README.txt：针对Maven版本，第三方软件等简要介绍

## 环境配置
1. 配置`MAVEN_HOME`环境变量
2. 配置`%MAVEN_HOME%\bin`到PATH
3. 命令行窗口测试`mvn -v`指令（查看maven的版本信息）

## 功能配置
修改`%MAVEN_HOME%/conf/settings.xml`配置文件

### 配置本地仓库位置
默认值是`~/.m2/repository`

### 配置阿里云镜像
```xml
<mirror>
    <id>alimaven</id>
    <name>aliyun maven</name>
    <url>http://maven.aliyun.com/nexus/content/groups/public/</url>
    <mirrorOf>central</mirrorOf>
</mirror>
```

## IDEA配置
Settings > Build, Exection, Deplopyment > Build Tools > Maven：
* Maven home path：[环境配置](#环境配置)中的MAVEN_HOME
* User settings file：[功能配置](#功能配置)中的settings.xml
